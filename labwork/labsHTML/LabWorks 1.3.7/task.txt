Create a page according to a mock-up.

Requirements: 

You need to create a <div> element -container, and 4 <div> elements inside - columns. 
The task is to paint them in grey/darkgrey colors one by one. The problem is that they should be all 100% height.
 Setting 100% to height property - won't work. 
One of the decisions is to use gradients - see example here - http://codepen.io/mariemosley/pen/01f15f7a2dfcff438d17cd77c298b710


Создайте страницу в соответствии с макетом.

Требования:

Вам нужно создать элемент <div> -container и 4 элемента <div> внутри - столбцы. 
Задача состоит в том, чтобы покрасить их в серый / темно-серый цвета один за другим.
 Проблема в том, что они должны быть все 100% высоты.
 Установка 100% для свойства height - не сработает. Одним из решений является использование
 градиентов - см. Пример здесь - http://codepen.io/mariemosley/pen/01f15f7a2dfcff438d17cd77c298b710
